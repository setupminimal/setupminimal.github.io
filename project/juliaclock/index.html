JuliaClock

September 12, 2017
project art school project

I created JuliaClock during High School for my required art credit. JuliaClock is a program that takes the current time, generates a unique Julia Set, and then simulates painting it.

If you are interested in the Source Code, you may be able to get JuliaClock to run on your computer. It relies on the Python Image Library and on TK. If you’d rather just look at pretty pictures: (and videos, at the bottom)
