Paperclip Optimizer: The Paperclip Maximization Game.

Paperclip Optimizer is a game about making paperclips. With a real simulated economy, many tools to enable paperclip production, and a charming green color-scheme, I'm honestly surprized that this game hasn't been nominated for any awards.

OpenStates.org-Powered Bill (and Law) Search

The OpenStates.org-Powered Bill (and Law) Search exists because while OpenStates is a great project, I wanted a simple, uncluttered interface to let me know about upcoming laws concerning particular topics from across the United States. I hope you find it useful as well.

Software Which I Have Written, but Which is not Polished:

Cambast

A cambast is a money lender. This project is a piece of scheduling software, based on the idea that every task has a certain value. It was inspired by "The Cambast and Lord Iron", a very good short story.

Julia Clock

Julia Clock is an experiment in beautiful time. It takes the current time and generates a unique image of a Julia Set, the colors, symmetries, position, rotation, size, and texture of which are all indications of different aspects of the time. Try watching it for a while, and see what you find!
