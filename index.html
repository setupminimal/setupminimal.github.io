Software Which I Have Written, but Which is not Polished:

Cambast

A cambast is a money lender. This project is a piece of scheduling software, based on the idea that every task has a certain value. It was inspired by "The Cambast and Lord Iron", a very good short story.

Julia Clock

Julia Clock is an experiment in beautiful time. It takes the current time and generates a unique image of a Julia Set, the colors, symmetries, position, rotation, size, and texture of which are all indications of different aspects of the time. Try watching it for a while, and see what you find!
